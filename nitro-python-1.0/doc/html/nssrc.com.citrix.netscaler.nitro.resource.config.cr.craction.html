craction module¶

class craction¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for cache redirection action resource.

name¶

Name of the action for which to display detailed information.<br/>Minimum length = 1.

crtype¶

Type to decide where to redirect the requests if the cache redirection policy is hit. The valid options are as follows: *CACHE - Directs all the requests to the cache if cache redirection policy is hit. *ORIGIN - Directs all requests to the origin server if the cache redirection policy is hit.<br/>Possible values = origin, cache.

builtin¶

Flag to determine whether CRACTION is default or not.<br/>Possible values = MODIFIABLE, DELETABLE, IMMUTABLE, PARTITION_ALL.

isdefault¶

A value of true is returned if it is a default cr action.

hits¶

The number of times the action has been taken.

referencecount¶

The number of references to the action.

undefhits¶

The number of times the action resulted in UNDEF.

comment¶

Comment. Any type of information about this responder action.
