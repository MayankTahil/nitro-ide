The name of the rdp server profile.<br/>Minimum length = 1<br/>Maximum length = 32.

rdpip¶

IPv4 or IPv6 address of RDP listener. This terminates client RDP connections.<br/>Minimum length = 1.

rdpport¶

TCP port on which the RDP connection is established.<br/>Default value: 3389<br/>Minimum length = 1<br/>Maximum length = 65535.
