S¶

Print a summary of how many probes were not answered for each hop.

n¶

Print hop addresses numerically instead of symbolically and numerically.

r¶

Bypass normal routing tables and send directly to a host on an attached network. If the host is not on a directly attached network, an error is returned.

v¶

Verbose output. List received ICMP packets other than TIME_EXCEEDED and UNREACHABLE.

M¶

Minimum TTL value used in outgoing probe packets.<br/>Default value: 1<br/>Minimum length = 1<br/>Maximum length = 255.

m¶

Maximum TTL value used in outgoing probe packets. For Nitro API, default value is taken as 10.<br/>Default value: 64<br/>Minimum length = 1<br/>Maximum length = 255.

P¶

Send packets of specified IP protocol. The currently supported protocols are UDP and ICMP.<br/>Minimum length = 1<br/>Maximum length = 20.

p¶

Base port number used in probes.<br/>Default value: 33434<br/>Minimum length = 1<br/>Maximum length = 65535.

q¶

Number of queries per hop. For Nitro API, defalut value is taken as 1.<br/>Default value: 3<br/>Minimum length = 1<br/>Maximum length = 65535.

s¶

Source IP address to use in the outgoing query packets. If the IP address does not belong to this appliance, an error is returned and nothing is sent.<br/>Minimum length = 1.

T¶

Traffic Domain Id.<br/>Minimum length = 1<br/>Maximum length = 4094.

t¶

Type-of-service in query packets.<br/>Maximum length = 255.

w¶

Time (in seconds) to wait for a response to a query. For Nitro API, defalut value is set to 3.<br/>Default value: 5<br/>Minimum length = 2<br/>Maximum length = 86399.

host¶

Destination host IP address or name.<br/>Minimum length = 1<br/>Maximum length = 64.

packetlen¶

Length (in bytes) of the query packets.<br/>Default value: 44<br/>Minimum length = 44<br/>Maximum length = 32768.
