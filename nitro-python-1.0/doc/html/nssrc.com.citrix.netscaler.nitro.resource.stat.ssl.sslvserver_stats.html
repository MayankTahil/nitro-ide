Statistics for SSL virtual server resource.

Name of the SSL virtual server for which to show detailed statistics.<br/>Minimum length = 1.

clearstats¶

Clear the statsistics / counters.<br/>Possible values = basic, full.

Rate (/s) counter for ssltotclientauthsuccess.

Number of hw decrypted bytes per SSL vserver.

Number of new sessions created.
