class ipsecalgprofile¶

Bases: nssrc.com.citrix.netscaler.nitro.resource.base.base_resource.base_resource

Configuration for IPSEC ALG profile resource.

name¶

The name of the ipsec alg profile.<br/>Minimum length = 1<br/>Maximum length = 32.

ikesessiontimeout¶

IKE session timeout in minutes.<br/>Default value: 60<br/>Minimum length = 1<br/>Maximum length = 1440.

espsessiontimeout¶

ESP session timeout in minutes.<br/>Default value: 60<br/>Minimum length = 1<br/>Maximum length = 1440.

espgatetimeout¶

Timeout ESP in seconds as no ESP packets are seen after IKE negotiation.<br/>Default value: 60<br/>Minimum length = 30<br/>Maximum length = 1200.

connfailover¶

Mode in which the connection failover feature must operate for the IPSec Alg. After a failover, established UDP connections and ESP packet flows are kept active and resumed on the secondary appliance. Recomended setting is ENABLED.<br/>Default value: ENABLED<br/>Possible values = ENABLED, DISABLED.
